Privacy

Browser

We want to set browser to open in privacy mode by default, and say with Google as homepage.
Right click Browser Icon -> Right click Browser Name -> Click Properties -> In tab "Shortcut", field "Target", add codes at the end.
Chrome: "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" " -incognito" "https://www.google.com"
Edge: "C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe" --profile-directory=Default -inprivate "https://www.google.com"

DNS

Domain Name System (DNS) translates website domains (e.g. google.com) into IP addresses for machine reading. For browsing privacy and security, we want use a trustable DNS service provider, say Cloudfare.
Cloudfare Setup Guide Windows: Settings -> Network & Internet -> Status -> Change adapter options -> Choose netowrk -> Click "Properties" -> Select "Internet Protocol Version 4 (TCP/IPv4)" -> Click "Properties" -> Choose "Use the following DNS server addresses" and type 1.1.1.1 and 1.0.0.1
Android: Settings -> Network & Internet -> Advanced -> Private DNS -> Choose "Private DNS provider hostname" and type "1dot1dot1dot1.cloudflare-dns.com"
