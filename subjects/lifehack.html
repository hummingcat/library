Privacy

Browser

We want to open the browser in privacy mode by default, and say set Google as homepage.
Windows: right clicck Browser Icon -> right click Browser Name -> Properties -> tab "Shortcut", field "Target", add codes at the end
- Chrome: "C:\...\chrome.exe" " -incognito" "https://www.google.com"
- Edge: "C:\...\msedge.exe" --profile-directory=Default -inprivate "https://www.google.com"
Android: long press App Icon -> long press "Incognito / New InPrivate Tab" -> drag new App Icon to suitable place

DNS

Domain Name System (DNS) translates website domains (e.g. google.com) into IP addresses for machine reading. For browsing privacy and security, we want to use a trustable DNS service provider, say Cloudfare.
Cloudfare Setup Guide
Windows: Settings -> Network & Internet -> Status -> Change adapter options -> choose network -> click "Properties" -> select "Internet Protocol Version 4 (TCP/IPv4)" -> click "Properties" -> choose "Use the following DNS server addresses" and type 1.1.1.1 and 1.0.0.1
Android: Settings -> Network & Internet -> Advanced -> Private DNS -> choose "Private DNS provider hostname" and type "1dot1dot1dot1.cloudflare-dns.com"
